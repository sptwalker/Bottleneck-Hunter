## 核心方法论：Serenity "三步法"

### 第一步：产业链拆解
- 用AI深度拆解产业链三层以上
- 从终端产品（如GPU）向上追溯，找出所有关键零部件及其原材料和设备

### 第二步：供应商检索
- 沿每个环节用AI检索供应商
- 重点筛选市值小、被机构忽视但技术独特的公司

### 第三步：交叉验证
- 用多个AI大模型从反面角度反复拷问投资逻辑
- 稀缺性是否唯一、技术是否会被替代、产能是否充足、客户是否已验证

## 投资哲学：投资瓶颈，而非热点

避开热门GPU公司，专攻支撑GPU运转但更稀缺的环节。光模块、激光器、衬底等，物理上不可替代且供需缺口巨大。

## 典型案例

- **通美晶体**：垄断95%磷化铟市场，涨10倍
- **西福斯半导体**：涨19.6倍
- **应用光电**：涨8.4倍
- 23支持仓股全部翻倍，平均涨幅超10倍

## A股适用性

中国是全球制造业最强国，在AI、人形机器人、商业航天等产业浪潮中，许多"卡脖子"零部件生产环节在中国。A股同样适用此方法论。
