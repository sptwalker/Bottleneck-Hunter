# 市场关注度 v2 公式 & Alpha 因子文档

> 版本: v2.0 | 更新日期: 2026-06-21

## 1. 市场关注度（market_attention）

### 1.1 五维加权公式

```
raw = 市值分 × 0.15 + 分析师覆盖分 × 0.20 + 成交量动量分 × 0.25 + 近3月涨幅分 × 0.15 + 机构持仓分 × 0.25
```

### 1.2 各维度分档表

**维度 1: 市值规模（权重 15%）**

| A 股（亿元） | 得分 | 美股（$B） | 得分 |

| < 30  | 1 | < 1   | 1 |
| < 100 | 3 | < 5   | 3 |
| < 300 | 5 | < 20  | 5 |
| < 1000| 7 | < 100 | 7 |
| ≥ 1000| 9 | ≥ 100 | 9 |

**维度 2: 分析师/机构覆盖（权重 20%）**

|---|---|
| < 3  | 1 |
| < 8  | 3 |
| < 15 | 5 |
| < 25 | 7 |
| ≥ 25 | 9 |

|---|---|

**维度 3: 成交量动量（权重 25%）**

`volume_ratio = 过滤后10日均量 / 60日均量`

| < 0.8 | 3 |

连续放量加分: 若最近 10 日内存在连续 3 日成交量 > 60 日均量 × 1.3，得分 +1（上限 9）。

**维度 4: 近 3 月涨幅（权重 15%）**

**维度 5: 机构持仓（权重 25%）**

| 持仓占比 (%) | 得分 |
|---|---|
| < 5   | 1 |

### 1.3 特殊调整规则

1. **成交量异常值过滤**: 计算 10 日均量前，对每日成交量检查: 若 `day_vol > 60日均量 × 10`，则将其截断为 `60日均量 × 3`。避免大宗交易或突发事件导致量比虚高。

2. **连续放量加分**: 最近 10 日内存在连续 3 天成交量 > 60 日均量 × 1.3 → 成交量维度得分 +1（上限 9）。

3. **IPO 加分**: 上市时间 < 365 天 → 总分 +2。新股信息传播不充分，存在额外信息差。

4. **最低关注度限制**: `market_attention = max(2.0, raw + ipo_bonus)`，所有上市公司至少有 2.0 的关注度基线。

5. **上限 10.0**: `market_attention = min(10.0, ...)`。

### 1.4 信息差

```
information_gap = 10.0 - market_attention
```

## 2. Alpha 因子（alpha_score）

### 2.1 基础公式

```
base_alpha = bottleneck_score × (1 - market_attention / 10)
alpha = clamp(base_alpha × urgency_multiplier × volume_price_discount + trend_bonus + smart_money_bonus, 0, 10)
```

### 2.2 量价背离折扣

当同时满足以下两个条件时，应用 ×0.8 折扣:
- 近 1 月涨幅 > 20%
- 成交量动量 (volume_ratio) < 0.8

含义: 价格上涨但成交量萎缩，可能是假突破，降低 Alpha 估值。

### 2.3 趋势加分（trend_bonus）

基于 `FinancialTrend` 中的加速度指标:
- 营收加速 > 0 且毛利扩张 > 0 → 最高 +1.5
- 连续增长季度数奖励
- 负面趋势 → 最低 -1.0

### 2.4 聪明钱加分（smart_money_bonus）

基于 `SmartMoneySignal`:
- 机构增持 / 北向资金净流入 / 龙虎榜买入 → 最高 +1.5
- 反向信号 → 最低 -1.0

### 2.5 紧迫度乘数（urgency_multiplier）

基于 `CatalystTimeline`:
- 近期催化剂（1-2 季度内）→ ×1.2
- 远期或无催化剂 → ×0.8~1.0

---

## 3. 最终评分（final_score）

```
final_score = quality_score^0.4 × alpha_score^0.6
```

- `quality_score` = `overall_score`（五维 LLM 评分的加权平均）
- `alpha_score` = 上述 Alpha 因子
- 几何加权均值，偏重预期差（Alpha 权重 60%）

## 4. 数据源

### 美股 (yfinance)
| 字段 | API 来源 |
|---|---|
| 分析师数 | `info["numberOfAnalystOpinions"]` |
| 机构持仓 | `info["heldPercentInstitutions"]` × 100 |
| IPO 日期 | `info["firstTradeDateEpochUtc"]` |
| 日线数据 | `stock.history(period="6mo")` → Volume, Close |

### A 股 (AKShare)
| 字段 | API 来源 |
|---|---|
| 覆盖机构数 | `stock_research_report_em` → 近 6 月去重 `机构名称` |
| 日线数据 | `stock_zh_a_hist(period="daily", adjust="qfq")` → 成交量, 收盘 |

## 5. 验算案例（6 只美股）

| 公司 | 代码 | 市值($B) | 分析师 | 量比 | 3M涨幅 | 机构% | 关注度 | 瓶颈分 | Alpha |
|---|---|---|---|---|---|---|---|---|---|
| NVIDIA | NVDA | 3400 | 59 | 0.99 | +8% | 65% | 7.2 | 8.0 | 2.5 |
| TSMC (ADR) | TSM | 800 | 28 | 1.10 | +15% | 18% | 5.6 | 9.0 | 5.0 |
| Applied Materials | AMAT | 130 | 32 | 1.53 | +25% | 82% | 8.1 | 7.5 | 1.5 |
| Ouster | OUST | 1.5 | 5 | 1.80 | +121% | 45% | 4.9 | 8.5 | 5.4 |
| SiTime | SITM | 5 | 12 | 1.05 | +40% | 90% | 5.5 | 8.0 | 4.2 |
| Amkor Tech | AMKR | 6 | 10 | 0.85 | -5% | 72% | 4.5 | 7.0 | 4.5 |

验算要点:
- NVDA: 超大市值 + 高分析师覆盖 → 高关注度，低 Alpha
- TSM ADR: 机构持仓偏低（ADR 特点） → 关注度适中，高瓶颈分推升 Alpha
- OUST: 小市值 + 低分析师覆盖 → 低关注度，高 Alpha
- AMAT: 连续放量 + 高机构持仓 → 关注度最高
